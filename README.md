✅ Step 1: Download the .drawio file from GitHub

Open the file in GitHub.

Click the Raw button (top-right on the file view).

Your browser will show plain XML code.

Save it:

Windows/Linux → Press Ctrl + S

Mac → Press Cmd + S

Or Right-click → Save As…

In the dialog box:

File name → roboshop.drawio

Save as type → All Files

Make sure the extension is .drawio (not .txt or .xml).

✅ Step 2: Open the file in Draw.io (diagrams.net)

Go to 👉 https://app.diagrams.net/
.

Choose Device when it asks for storage.

Click File → Open From → Device.

Select your saved file: roboshop.drawio.

Now the full diagram will appear 🎉.

✅ Step 3: Export as an Image (PNG/JPEG)

In draw.io, go to File → Export As.

Click Export and save the image.

🔹 Extra: Alternative Methods

If you have the GitHub repo cloned locally:

Just save the file directly as roboshop.drawio and open it in Draw.io.

You can also drag-and-drop the .drawio file into draw.io
 directly.
